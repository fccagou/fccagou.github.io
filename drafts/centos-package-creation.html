Présentation

centpkg est utilisé pour le débveloppement des paquets Centos. Il s'interface par défaut avec git .centos.org

En l'installant dans un conteneur docker centos7, voici les commandes réalisées :

yum install -y epel-release 
yum install -y python-setuptools pyrpkg python-centos

git clone https://git.centos.org/r/centpkg.git

cd centpkg
python ./setup.py install

centpkg --help

# \o/

Pour aller plus loin

yum install -y python2-simplejson
git clone https://git.centos.org/r/centos-git-common.git
